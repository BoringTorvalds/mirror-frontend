`WeatherContainer` (component)
==============================

React component represents WeatherContainer
A container that maps to state.weather and display in Weather Component

Props
-----

### `current` (required)

An object of weather container current temperature

type: `object`


### `currentLocation`

Current location to look up weather

type: `string`
defaultValue: `null`


### `daily`

Next 7 days forcast of weather

type: `any`
defaultValue: `[]`


### `height`

Height of component

type: `string`
defaultValue: `null`


### `iconType`

Icon type of weather

type: `string`
defaultValue: `null`


### `isFetched`

defaultValue: `false`


### `isFetching`

defaultValue: `false`


### `summary`

Summary of current weather

type: `string`
defaultValue: `null`


### `temparature`

defaultValue: `null`


### `width`

Width of component

type: `string`
defaultValue: `null`
